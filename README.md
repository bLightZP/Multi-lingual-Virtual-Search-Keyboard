# Multi-lingual-Virtual-Search-Keyboard

Designed for Delphi 7, this minimal-dependency (only GDI+) virtual keyboard.

Integrates with layered forms and GDI+ to render elements with transparency effects (see screenshot).

Automatically detects installed keyboard language layouts and provides a virtual-key to switch between language layouts.

Supports Shift/Symbol layouts similar to mobile phone keyboards.

Works with both physical keyboard input and mouse input.

Text-input field supports select, cut, copy and paste operations similar to TEdit.

Fully unicode complient, even shows Emoji (not erased perfectly).
